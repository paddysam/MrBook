# 《乌龙老板》

## 📖 项目简介

一个农村老实人，单亲家庭长大，拼尽全力考上二本的程序员毕业生，初入社会因为笨和粗心做错很多事情，但做错的事情越多反而赚越多钱的反转爽文故事。

**核心主题**：傻人有傻福，塞翁失马焉知非福

---

- **主线**：乌龙发财（笨和粗心导致失误→运气好反转成功）

- **感情线**：王朴抛弃林静→被美女害惨→醒悟挽回
- **暗线**：生父暗中帮助→相认→父亲破产→王朴救父亲

## 🎯 故事核心

### 主角特点
- **笨和粗心**：导致各种失误
- **运气逆天**：每次失误都反转成功
- **真诚善良**：王宝强式的憨厚

### 四条故事线

**主线：乌龙发财**
- 面试失误→意外入职
- 工作失误→意外发现问题
- 创业失误→意外踩中风口
- 投资失误→意外买对股票
- 合同失误→意外获得资产

**支线：打脸爽文**
- 回村打脸：第一次有钱买车回村（第33章）
- 时机：前期，刚赚到100万
- 对象：势利眼邻居李富贵夫妇
- 只此一次，不重复

**感情线：辜负与回归**
- 初期：林静完美女友
- 中期：王朴变心抛弃林静
- 低谷：被美女害惨
- 醒悟：林静才是最好的
- 挽回：真诚忏悔重新在一起

**暗线：父子故事**
- 前90章：一切看起来是运气
- 第95章：揭秘，原来是父亲在帮
- 第97章：父亲破产
- 第100章：王朴又犯错了
- 第101章：失误救了父亲，王朴成超级富豪
