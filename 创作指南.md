# 《乌龙老板》创作指南

## 🎯 核心原则（时刻牢记）

### 主题：傻人有傻福

- 王朴因为**笨**和**粗心**导致失误
- 每次失误都**运气好**反转成功
- 不是聪明，是运气（直到揭秘是父亲帮忙）

### 三个"不要"

1. ❌ **不要让王朴变聪明**：他就是笨，一直笨，靠运气不靠智商
2. ❌ **不要写技术细节**：避免专业术语，通俗化描写
3. ❌ **不要让林静有虚荣心**：她是完美女友，不在乎钱

### 🔴 第四个"不要"：最最重要！

**不要"告诉"，要"展示"**

**永远记住**：
- ❌ 不要告诉读者这个人是什么
- ❌ 不要告诉读者剧情是什么  
- ❌ 不要做总结
- ❌ 不要透剧

**只做这些**：
- ✅ 写氛围
- ✅ 写动作
- ✅ 写情感
- ✅ 让读者自己看，自己感受，自己判断

**错误示例**：
```
王朴是个很笨的人，但他运气很好。（× 这是在告诉读者）
林静很爱他，不在乎钱。（× 这是在总结）
这次失误会让他发财。（× 这是在透剧）
```

**正确示例**：
```
王朴又把文件发错了，手忙脚乱地想撤回。（✓ 展示笨）
林静看着只剩100块的钱包，笑着说："够了，我们一起省着点花。"（✓ 展示不在乎钱）
他签下合同，手有点抖。完了，这下真的被坑了。（✓ 不透剧）
```

### ⚠️ 最重要：故事线穿插原则

**🔴 绝对不要独立写线！**

❌ **错误写法**：

- 第1-30章：只写主线（赚钱）
- 第31-40章：只写支线（打脸）
- 第41-50章：只写感情线
- 第51-60章：只写暗线
- **这样像是完成任务，不是讲故事！**

✅ **正确写法**：

- 每一章都是**主线推进时，顺便带出其他线**
- 🎯 主线到某个剧情点 → 💪 方便加支线 → 就加支线
- 🎯 主角遇到困难 → ❤️ 女友自然出现安慰 → 感情线推进
- 🎯 主角要破产 → 🔮 暗中有人帮 → 埋暗线伏笔
- **四条线交织在一起，你中有我，我中有你**

**例子1：第1章（村里的穷小子）**

- 🎯 主线：介绍王朴背景（主）
- 💪 支线：邻居嘲讽，铺垫打脸（插入）
- ❤️ 感情线：林静陪他，不嫌弃（插入）
- **一章包含3条线，自然穿插**

**例子2：第6章（第一次工作乌龙）**

- 🎯 主线：发错邮件，犯错（主）
- 💪 支线：同事嘲笑（见缝插针）
- ❤️ 感情线：沮丧回家，女友安慰（顺便）
- **主线推进中，其他线自然出现**

**例子3：第19-21章（回村打脸）**

- 第19章：💪 重点写邻居恶心（支线为主）+ 🎯 王朴有钱了（主线）
- 第20章：🎯 开车回村（主线）+ ❤️ 林静陪同（感情线）
- 第21章：💪 打脸高潮（支线）+ ❤️ 母亲喜欢林静（感情线）
- **虽然这几章打脸是重点，但还是混合其他线**

**写作时问自己**：

- ❓ 这章只写了一条线吗？ → 加入其他线
- ❓ 这个情节能不能顺便推进感情线？ → 加上
- ❓ 主角成功了，有没有人来巴结？ → 加支线打脸
- ❓ 这次运气太好了，要不要埋暗线？ → 加一句伏笔

## 👤 人物核心（写作时参考）

### ⚠️ 视角使用原则（非常重要！）

**写主角时：必须用主角视角**

✅ **正确写法**（主角视角）：

```
王朴看着面试官冷漠的眼神，心里一沉，完了，这次又砸了。
他抓了抓头发，结结巴巴地说："对...对不起，我刚才说错了..."
```

❌ **错误写法**（上帝视角）：

```
王朴不知道，这次失误其实是个机会。隔壁小公司老板正好听到，觉得他很真诚。
（这是剧透！不要告诉读者后面会发生什么！）
```

✅ **正确写法**（不剧透）：

```
王朴垂头丧气地走出面试室，觉得自己完蛋了。
（读者和王朴一样，以为失败了，后面反转才有惊喜）
```

**写环境时：可以用上帝视角烘托氛围**

✅ **正确写法**（环境描写）：

```
夏天的杭州，闷热潮湿，知了在树上聒噪地叫着。
地下室里没有空调，只有一台破旧的电风扇吱呀吱呀地转着。
墙角堆着几个纸箱，里面是王朴的全部家当。
（烘托贫穷、压抑的氛围，不剧透）
```

❌ **错误写法**（环境描写中剧透）：

```
地下室虽然破旧，但王朴不知道，他很快就会搬出去，住上大别墅。
（不要在环境描写里剧透！）
```

**⚠️ 最重要：不要"告诉"，要"展示"**

❌ **错误写法**（直接告诉）：

```
王朴觉得这一切很不真实。
他怀疑自己是不是在做梦。
他很迷茫，不知道为什么会成功。
（这是在"告诉"读者，太直白）
```

✅ **正确写法**（通过展示）：

```
王朴看着手中的酒杯，发了会儿呆。
周围的笑声仿佛隔了一层玻璃，听不真切。
他喝了一口红酒，却尝不出味道。
（通过动作、感官展示，让读者自己感受到他的迷茫）
```

**不要写的词句**：

- ❌ "他觉得不真实"
- ❌ "这一切是真的吗"
- ❌ "他很迷茫"
- ❌ "他不知道为什么"
- ❌ "他怀疑..."
- ❌ "他感到..."

**要用的方式**：

- ✅ 动作：发呆、眨眼、目光涣散、打寒颤
- ✅ 感官：听不清、尝不出味道、看不清、触感麻木
- ✅ 对比：周围热闹但他疏离、声音模糊、好像隔着玻璃

**写作原则**：

- 把环境和动作写出来就好
- 是什么感受，让读者自己去体会
- 不要直接说出来

**总结**：

- 🎯 **写王朴的想法、感受、行动**：用王朴视角，读者和他一起体验
- 🌍 **写场景、环境、氛围**：可以客观描述，烘托情绪
- ❌ **绝不剧透**：不要告诉读者后面会发生什么，保持悬念
- ❌ **不要上帝视角**：不要写"王朴不知道的是..."、"其实..."、"殊不知..."
- ❌ **不要直接告诉**：不要写"他觉得"、"他感到"，要通过动作、感官展示

**例子对比**：

**场景：王朴签了"垃圾合同"**

❌ **错误**（上帝视角+剧透）：

```
王朴签下了合同，以为自己被坑了。
但他不知道，这块地半个月后就要拆迁，他即将暴富。
（剧透了！反转没惊喜了！）
```

✅ **正确**（主角视角+不剧透）：

```
王朴看着合同上的字，手有点抖。
"这...这真的没问题吗？"他抬头看向老板。
老板摆摆手："放心，这块地虽然偏，但总归是值点钱的。"
王朴咬咬牙，在合同上签了自己的名字。
他心想：完了，这下真的被坑了。
（读者和王朴一样担心，后面反转才爽）
```

### 王朴（参考：王宝强）

> 详见：`人物确定版.md`

**必须保持的特质**：

- 憨厚老实（像王宝强）
- 有点笨（脑子转得慢）
- 粗心大意（看错、听错、记错）
- 笨手笨脚（摔东西、碰东西）
- 不善言辞（说话磕巴）
- 运气逆天（每次失误都反转）

**口头禅**：

- "不好意思，我...我搞错了"
- "啊？你说啥？"
- "对不起对不起..."

**习惯动作**：

- 抓头发、挠头、摸鼻子
- 傻笑、手足无措

## 📖 四条故事线（写作框架）

### 主线：乌龙发财

> 详见：`章节规划.md`

**写作要点**：

- 每5-8章一个完整的"失误-反转"循环
- 失误要因为**笨**或**粗心**
- 反转要意外、震撼
- 王朴每次都很懵："我怎么又成功了？"

**典型模式**：

```
王朴因为笨/粗心出错
→ 所有人觉得他完了
→ 意外发生
→ 失误反而是对的
→ 王朴成功（自己都懵）
```

### 支线：回村打脸势利眼

> 详见：`章节规划.md` 第31-33章

**只有一次打脸**：

- 第33章：第一次有钱，买车回村打脸
- 时机：主角刚赚到100万的时候
- 地点：老家村里

**打脸对象**：李富贵夫妇（势利眼邻居）

- 以前嘲讽："没爹的孩子没出息"
- 现在：震惊、后悔、想攀附

**写作要点**：

- 只写这一次打脸，后面不再重复
- 重点在对比反差和母亲扬眉吐气
- 点到为止，不拖沓
- 体现主角孝顺（第一时间想到母亲）

### 感情线：辜负与回归

> 详见：`感情线详细设定.md`

**关键节点**：

- 1-30章：林静无私付出
- 31-50章：王朴变心
- 58-60章：王朴提分手
- 61-65章：被美女害惨
- 71-75章：醒悟，林静最好
- 76-85章：求助林静，并肩作战
- 86-95章：追妻挽回
- 96-100章：重新在一起

**重要场景**：详见感情线文档的46个场景

### 暗线：父子故事

> 详见：`生父暗线详细设定.md`

**关键节点**：

- 1-94章：王朴以为是运气好
- 95章：生父沈千帆现身，揭秘身份
- 96章：王朴愤怒，父子冲突
- 97章：父亲要破产（反转）
- 98-99章：王朴决定帮助
- 100章：又犯错了
- 101章：失误救了父亲，王朴成超级富豪
- 103章：父子和解

**绝对不要**：前94章不要有神秘人出现！

## 🚫 绝对避免的错误

### 人物方面：

1. ❌ **不要让王朴变聪明**
   - 他就是笨，一直笨到底
   - 成功靠运气，不是变聪明了

2. ❌ **不要让林静有虚荣心**
   - 她不在乎钱，只在乎人
   - 被抛弃也不怨恨
   - 是完美女友

3. ❌ **不要让王朴装傻**
   - 他是真的笨，不是装的
   - 失误是真的失误，不是计谋

### 剧情方面：

4. ❌ **不要提前暴露父亲**
   - 前94章完全不提神秘人
   - 只说运气好
   - 不要有任何暗示

5. ❌ **不要写太多技术**
   - 避免前端、后端、框架等术语
   - 用"写代码、做网站"代替
   - 重点是失误，不是技术

6. ❌ **不要让失误太离谱**
   - 失误要在合理范围内
   - 粗心的人确实会犯的错
   - 不要为了搞笑而搞笑

7. ❌ **不要"告诉"，要"展示"**（最重要！）
   - 不要写"他觉得不真实"
   - 不要写"这一切是真的吗"
   - 不要写"他很迷茫"、"他怀疑"
   - 不要写"他很笨"、"他很幸运"
   - 不要总结、不要评价、不要透剧
   - 只写氛围、动作、情感、故事
   - 让读者自己感受、自己判断

## ✅ 应该强化的元素

### 1. 王朴的笨和粗心（喜剧来源）

- 多写他的笨拙表现
- 多写他的粗心失误
- 多写他的反应慢
- 制造喜剧效果

### 2. 失误-反转（核心卖点）

- 失误过程要详细
- 所有人都以为完了
- 反转要意外震撼
- 王朴自己都懵

### 3. 打脸场景（爽点来源）

- 势利眼的嘲讽要详细
- 打脸的过程要细致
- 势利眼的震惊和尴尬要写足
- 对比要强烈

### 4. 感情线的虐与甜

- 前期林静的好要写足
- 中期王朴的渣要写狠
- 分手要虐心
- 挽回要感人

## 📊 节奏控制

### 每章结构建议：

- **开头**：简短回顾或新场景
- **中间**：冲突/失误/对话
- **结尾**：小高潮/悬念/反转
- **字数**：完全自由发挥，不限制

### 整体节奏：

- 每5-8章：一个完整的"失误-反转"
- 每10-15章：一次大打脸
- 每25-30章：一个大高潮
- 每卷结尾：重大转折
- **重要情节可以多写几章，平淡过渡可以简短**

- 林静温柔："没事的，我们慢慢来"

### 描写（适量）

- 动作描写：王朴抓头发、傻笑
- 环境描写：简单几句带过
- 心理描写：简短直白

### 叙事（简洁）

- 不要大段叙述
- 多用对话推进剧情
- 节奏要快

## 🎬 每章写作流程

### 特别提醒：第1章倒序钩子

**第1章非常重要！**

**写法**：倒序开头（先写未来成功的场景）

- 场景：天台派对，主角已经很成功
- 内容：纸醉金迷，美女环绕，奢华生活
- 钩子：主角自己都疑惑"我怎么走到今天的？"
- 断章：留下悬念，不解释

**作用**：

1. 吸引读者：穷小子怎么变富豪的？
2. 制造反差：第2章拉回地下室的贫穷现实
3. 埋下悬念：为什么连他自己都不知道怎么成功的？

**第2-3章**：正序写背景，重点铺垫势利眼邻居

**第4章开始**：主线正式展开
