# 第012章：订单暴涨（框架）

## 章节定位
- **类型**：反转爽文章
- **时间线**：紧接第011章，第二天上午
- **场景**：办公室（绝望等待）→ 电话来了（反转）→ 办公室沸腾（庆祝）
- **核心作用**：危机反转成功，王朴再次化险为夷，张伟再次被打脸
- **节奏**：压抑→突转→爆发

---

## 结构：三幕剧

### 第一幕：绝望的早晨（25%）
- 王朴来上班，准备被开除
- 同事们看他的眼神复杂
- 张伟冷笑等着看好戏
- 王朴坐立不安

### 第二幕：李总来电（35%）
- 张建国办公室电话响了
- 华星科技李总来电
- 追加10倍订单，急需大量配件
- 发现王朴多订的货正好派上用场
- "你们是唯一有现货的！"

### 第三幕：反转庆祝（40%）
- 张建国宣布好消息
- 办公室瞬间沸腾
- 王朴懵了，不敢相信
- 张伟脸色铁青，一句话说不出来
- 同事们围着王朴庆祝
- 暗线伏笔：李总提到"有朋友推荐"

## 第一幕：绝望的早晨（25%笔墨）

### 场景：办公室，周四早上9点

**王朴来上班**：
- 王朴走进办公室
- 昨晚几乎没睡
- 脸色苍白，眼睛红肿
- 他做好了被开除的准备

**同事们的反应**：
- 办公室里的人看到他
- 气氛有点尴尬
- 小马走过来："兄弟，怎么样了？"
- 王朴摇头："不知道，还没通知"
- 李姐叹气："唉..."

**张伟的冷笑**：
- 张伟坐在座位上
- 看到王朴，嘴角带着冷笑
- "还敢来上班啊"
- 小声嘀咕，但王朴听到了
- 王朴没理他，走到自己座位

**坐立不安**：
- 王朴坐下来
- 打开电脑
- 但完全看不进去
- 手放在键盘上，一动不动
- 满脑子都是昨天的事
- 180万
- 他完了

**张建国办公室的门**：
- 王朴不时看向张建国的办公室
- 门关着
- 不知道张总在里面做什么
- 是不是在准备开除通知？
- 还是在想办法处理那批货？

**林静的消息**：
- 手机震动
- 林静发来消息："还好吗？"
- 王朴回："还没说"
- "别担心，会有办法的"
- 王朴看着这条消息
- 苦笑
- 会有什么办法呢？

### 张伟等着看好戏

**张伟的心理**：
- 张伟假装工作
- 实际上一直在观察王朴
- 心里很爽
- 终于
- 这次王朴真的完了
- 180万，这次看他怎么办

**跟小周说话**：
- 张伟转头对小周说
- "180万啊，公司肯定拿不出来"
- "王朴估计要赔钱了"
- "赔一辈子都赔不起"
- 小周不好意思接话
- 只是"嗯"了一声

## 第二幕：李总来电（35%笔墨）

### 张建国办公室的电话响了

**电话铃声**：
- 突然
- 张建国办公室的电话响了
- 铃声在安静的办公室里格外刺耳
- 王朴下意识看了过去
- 透过玻璃，看到张建国接起电话

**张建国的表情**：
- 张建国接电话
- 先是皱眉
- 然后愣住了
- 接着眼睛瞪大
- 最后站了起来
- 所有人都注意到了

**办公室的注意力**：
- 大家都停下手里的活
- 看向张建国的办公室
- 发生什么事了？
- 张总的表情好像...很激动？

### 李总的电话

**张建国打开门**：
- 张建国推开门
- 走出来
- 脸上带着笑容
- 但还有点不敢相信的表情
- "王朴，过来一下"

**王朴紧张**：
- 王朴的心一沉
- 来了
- 要开除他了
- 他站起来
- 腿都有点软
- 走到张建国面前
- "张总..."

**张建国的话**：
- "刚才华星科技的李总来电"
- 王朴低着头
- 等着被训斥
- "他们要追加订单"
- 什么？
- 王朴愣了一下

**追加订单**：
- "李总说，他们决定把项目规模扩大10倍"
- "现在急需大量配件"
- "问我们能不能马上供货"
- 张建国看着王朴
- 眼神很复杂
- "我说，我们正好有大批现货"

**王朴懵了**：
- 王朴站在那里
- 完全没反应过来
- 追加订单？
- 10倍？
- 急需配件？
- 现货？

**张建国继续说**：
- "李总说，他们找了好几家供应商"
- "都说要等一个月才能交货"
- "只有我们有现货，能马上发货"
- 停顿了一下
- "所以他们决定全部从我们这里采购"

### 意识到真相

**张建国笑了**：
- 张建国看着王朴
- 突然笑了
- "小王啊"
- "你这运气..."
- "真是..."
- 他摇头，笑得很无奈

**办公室里的人**：
- 所有人都听到了
- 先是愣住
- 然后反应过来
- 小马第一个跳起来："什么？！"
- "订单？追加订单？"
- "那王朴的货..."

**张建国宣布**：
- 张建国转向大家
- "没错"
- "王朴多订的那批货"
- "正好用上了"
- "不但不是损失"
- "反而救了我们"

- "兄弟！你是锦鲤啊！"

**老陈难得开口**：
- 连老陈都摘下耳机
- 走过来
- "不错"
- 拍了拍王朴的肩膀
- 然后又走回去了

### 王朴的反应

**王朴还在懵**：
- 王朴站在那里
- 完全懵了
- 他看看张建国
- 又看看同事们
- 再看看自己的手
- 这...这是真的吗？

**难以置信**：
- "我...我没被开除？"
- 张建国笑了："开除？"
- "你现在是功臣"
- "功臣？"
- 王朴更懵了

**张建国解释**：
- "李总说，这批订单很急"
- "如果我们没有现货，他们就要找别人"
- "那我们就失去这个大客户了"
- "现在因为你多订的货"
- "我们成了唯一能马上供货的"
- "李总很满意"

**王朴眼眶红了**：
- 王朴听着
- 眼眶又红了
- 但这次不是因为绝望
- 是因为不敢相信
- "我...我真的不用被开除了？"
- "不用"
- 张建国笑着说
- "不但不用，还要奖励你"

### 张伟的反应

**张伟僵住了**：
- 只有张伟
- 坐在座位上
- 像被雷劈了一样
- 嘴巴半张着
- 一句话说不出来

**不可能**：
- 不可能
- 怎么可能
- 180万的损失
- 怎么就变成了救公司？
- 王朴这个笨蛋
- 怎么又走运了？

**内心崩溃**：
- 张伟的脸色铁青
- 昨天他还在幸灾乐祸
- 还在想着王朴要被开除了
- 结果
- 转眼就变成功臣了？
- 他不甘心
- 但又无话可说

**起身离开**：
- 张伟站起来
- "我去上个厕所"
- 声音很僵硬
- 走出办公室
- 脚步很快
- 像是在逃

### 暗线伏笔

**张建国提到一个细节**：
- "对了"
- 张建国突然想起来
- "李总在电话里说了句话"
- "他说，有朋友推荐他们扩大项目规模"
- "说现在是好时机"

**小马好奇**：
- "什么朋友？"
- "不知道"
- 张建国摇头
- "李总也没细说"
- "反正对我们是好事"

**王朴听到了**：
- 王朴听到这句话
- 心里有点疑惑
- 有朋友推荐？
- 什么朋友？
- 为什么这么巧？
- 但他没多想
- 现在满脑子都是劫后余生的庆幸

**小马提议**：
- "今晚要不要聚一下？"
- "庆祝一下？"
- "好啊好啊！"小刘附和
- 大家都笑了

**王朴给林静发消息**：
- 王朴拿出手机
- 给林静发消息
- "静静，我没事了"
- "真的？！"
- "嗯，订单追加了，我的货正好用上了"
- "太好了！"
- 王朴看着手机，笑了

**张建国的话**：
- "好了，都回去工作吧"
- 张建国拍拍手
- "王朴，你继续跟进这个订单"
- "好的，张总"
- 王朴回到座位
- 整个人还飘飘的
- 不敢相信这是真的

- 他坐在座位上
- 看着电脑屏幕
- 昨天还绝望得想哭
- 今天就成了功臣
- 这反转
- 也太快了吧
- 他不知道

## 写作注意事项

### ✅ 一定要做的

1. **反转要突然有力**：
   - 李总的电话要来得突然
   - 办公室从绝望到沸腾的转换要快
   - 对比要强烈

2. **王朴的反应要真实**：
   - 他懵了，不敢相信
   - 反复确认"真的不被开除了？"
   - 劫后余生的感觉

3. **张伟的打脸要爽**：
   - 从幸灾乐祸到傻眼
   - 一句话说不出来
   - 起身逃离现场

4. **暗线伏笔要埋**：
   - "有朋友推荐"这句话很关键
   - 不要说明白是谁
   - 让读者有悬念

5. **情绪要释放**：
   - 前一章压抑
   - 这一章爆发
   - 让读者和王朴一起松口气

### ❌ 绝对不要做的

1. **不要让反转太容易**：
   - 李总来电前，气氛要压抑
   - 王朴真的以为自己要完了
   - 不要提前剧透

2. **不要让王朴太聪明**：
   - 他不会想到订单会追加
   - 纯粹是运气
   - 他自己也懵

3. **不要把暗线说明白**：
   - 只提"有朋友推荐"
   - 不要说是谁
   - 为后面揭秘留空间

4. **不要让张伟当场崩溃**：
   - 他只是傻眼、难受
   - 但不会当场发作
   - 为后续继续对抗留空间
